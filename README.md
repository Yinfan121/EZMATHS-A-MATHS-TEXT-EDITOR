##

EZMATHS 是一个旨在利用 KaTeX 而成的简单快捷的数学文本编辑器。

# 使用方法
EZMATHS是一个Web软件,你可以直接打开源代码中的index.html文件。它可以保存打开及编辑专属于它的文档.ezdf文件。

[立即使用](https://yinfan121.github.io/EZMATHS-A-MATHS-TEXT-EDITOR/)
